++++++++++++++++++++++++++++++++++++++++++++++++
                INSTALLATION
++++++++++++++++++++++++++++++++++++++++++++++++

The following applies to the 'Default' folder:

Move TheSplit.vst3 to the following path:

/Library/Audio/Plug-Ins/VST3/

And move TheSplit.component to:

/Library/Audio/Plug-Ins/Components/

*NOTE: /Library/Audio/Plug-Ins/VST3/ is NOT
to be confused with /Library/Audio/Plug-Ins/VST/

If you have an older Mac or the previous installation
did not run properly, please repeat the installation
with the 'Compatible' folder.
